Bricks

A design system by MyOnlineStore

What is Bricks?

Bricks is a modular design system built primarily in React. It uses the same principles as building blocks to create an easy-to-use and fun e-commerce platform.

The foundations of our platform

The core of Bricks is built on a shared view of design rules and reasons reflected in functional and modular components. Together they create consistency across all applications of MyOnlineStore.
