Numbers

A number to indicate the amount of new items.

Usage and guidelines
  • Number badges indicate the number of new items
  • Contain only numbers
  • Use colors to indicate severity

Subscription badges

Short tags used to indicate availability status within our subscription model.

Usage and guidelines
  • Text badges can contain the following statuses: (NEW, BETA, PRO, PREMIUM)
  • Are always in caps
  • Use the color of the subscription level they represent otherwise yellow-base
  • Never have a background
Examples

NEW | BETA | PRO | PREMIUM
